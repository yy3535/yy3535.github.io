# 语法

1.标题

# 一级标题
## 二级标题
### 三级标题

2.列表

- 1
- 2
- 3
1. 列表1
2. 列表2
3. 列表3

3.引用

> 这是一个引用
>> 二级引用
>>> 三级引用

4.分割线

---

5.链接

[链接1](www.baidu.com)
[链接2]:www.baidu.com
这是链接：[链接2][链接1]

6.图片

![图片1](https://www......gif)
![图片1](./xxx.png)
[图片2]:https://www......jpg
这是图片：![图片1][图片2]

7.代码框

  • 所有的html标签标签必须用代码框框起来，少的用单行，多的用多行，否则都会报错。
单行：
	`<p>…</p>`
多行：
	```js/md/php等语言
	<p>…</p>
	```
