# github.io网站搭建

  1. 注册账号

  2. 创建一个新仓库，名称必须为：username.github.io，username是你的github用户名。

  3. 把该仓库克隆到本地，新建index.html,js,css,img。本地浏览后，git提交代码到仓库。
